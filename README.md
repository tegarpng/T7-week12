# Dashboard Penjualan Supermarket

Aplikasi dashboard visualisasi data penjualan supermarket berbasis PySide6. Pada tugas ini, mahasiswa diharapkan dapat mengimplementasikan library matplotlib, dan pandas 

## Dataset

**Sumber:** Supermarket Sales Dataset — Kaggle  
**Link:** https://www.kaggle.com/datasets/faresashraf1001/supermarket-sales
